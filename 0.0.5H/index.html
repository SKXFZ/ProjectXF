Codigo-Fonte

Comando para deixar a escrita em forma de codigos de programação que seria(code):
Ex da linguagem C 

             
int horas, jogados, inativo;
char Atividade[50];

printf("Digite quantas horas você esteve acordado durante o dia? ");
scanf("%d", &horas);
printf("Qual atividade você passou mais tempo fazendo? ");
fseek(stdin, 0, SEEK_END);
gets(Atividade);
printf("Quantas horas você passou praticando a atividade?");
scanf("%d", &jogados);
Inativo = horas - jogados;
printf("Você esteve acordado %d durante o dia praticando a atividade %s durante %d e ficou %d inativo.", horas, Atividade, jogados, Inativo);

Marcações de Texto

Para você deixar um texto marcado, basta usar a tag MARK.
Se quiser usar outra cor, use o style=background color e escolha a cor. Tanto nos texto especificos ou em todo o body, colocando o style na parte de head.

Textos grandes e pequenos

Use a tag big para aumentar o texto selecionado, porém a tag está obsoleto.

Use a tag small para diminuir o tamanho do texto selecionado.

Texto deletado e inserido

Para colocar um texto com um traço no meio, indicando que o mesmo não é considerado, coloque a tag del.
Para colocar uma sublinhação embaixo do texto, indicando uma enfase nele coloque a tag ins ou u(não semantico)

Textos sobrescrito e subscrito

Para colocar alguma coisa elevado, coloque a tag sup.
Para colocar alguma coisa rebaixado, use a tag sub.
